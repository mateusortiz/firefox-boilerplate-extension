# My Firefox Extension

> Clone and Make your extension on Firefox

## Setup

Install dependencies:

```sh
npm install
```

Compile:

```sh
npm start
```

## Debug

1. Go to `about:debugging`
2. `Load temporary Add-on`
3. Select the `manifest.json`
